Description :
Ce projet permet de gérer des comptes bancaires en utilisant une architecture de type client-serveur.
Les deux parties sont très proches et leur mise en fonctionnement est similaire.

 -> Le fichier makefile
make (exécuter le makefile)
make clean (nettoyer tout)
make clean_objs (nettoyer que les fichiers objets)

 -> Exécuter ./serveur pour démarrer la communication
 -> Exécuter ./client pour se connecter
